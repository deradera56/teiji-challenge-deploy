# 定時退社チャレンジ（α＋メタ要素）

『今日も18時に帰れるか？』縦持ちカジュアルゲーム。GDD v1 ベースの実装。

## 起動方法

1. Godot 4.3 以降を起動
2. インポート → このフォルダの `project.godot` を選択
3. F5（実行）

## 遊び方

- 9:00 出社。タスクカードが次々に発生する（実時間で1日≒3分）
- カードごとに **▶着手 / 🤖AI / ✕断る** を選ぶ。手作業は同時に1つだけ
- **17:30 に帰宅ラッシュ発生**。18:00までに全タスクを捌けば定時退社！（AI委任中・作業中のタスクは締切が進行しない）
- 集中・やる気・信用のどれかが0になると強制帰宅（失敗）
- ☕休憩で集中力回復（クールダウン60分）
- 稼いだ💰予算でAI導入・設備強化・転職（会社＝難易度変更）

## 構成（データ駆動）

| パス | 内容 |
|---|---|
| `data/tasks.json` | タスク定義。追記するだけで新タスク追加 |
| `data/events.json` | ランダムイベント（VPN障害、役員巡回など） |
| `data/companies.json` | 会社＝難易度（イベント率・報酬倍率） |
| `data/upgrades.json` | AI/設備アップグレードとランク定義 |
| `data/achievements.json` | 実績40種の定義（条件・XP・ティア） |
| `scripts/autoload/` | Config（データ読込）/ Meta（セーブ・実績判定・プレイヤーレベル）/ UiTheme |
| `scripts/game/` | コアループ（game.gd）とタスクカード |
| `scripts/ui/` | タイトル・会社選択・ショップ・リザルト・実績一覧 |

- UIは全てコード生成（画像素材不要）。日本語はOSフォントにフォールバック
- セーブ: `user://save.json`（自動保存）
- データ整合性チェック: `godot --headless -s tests/data_check.gd`

## 実績・プレイヤーレベル（中毒性のための成長要素）

- 定時退社回数・連続記録・熟練度・コンボ・レア引きなど生涯統計から実績40種を判定（`Meta.check_achievements()`）
- 実績解除でXPを獲得し、プレイヤーレベルが上昇（`Meta.player_level()`、カテゴリ別熟練度やランクとは独立した進行軸）
- タイトル画面に「次の実績まであと一歩」ヒントを表示し、ゴールの近さで継続動機を作る（ゴールグラディエント効果）
- リザルト画面でレベルアップ演出・実績解除演出を表示。専用の実績一覧画面（`Achievements.tscn`）で解除済み/未解除（進捗バー付き）を確認できる
